# 孙广亚的个人博客

一个关于学术研究的个人博客网站。

## 项目简介

这是一个静态个人博客，用于展示学术论文和个人信息。

## 页面结构

- **index.html** - 首页/主页
- **papers.html** - 发表论文列表
- **about.html** - 关于我
- **styles.css** - 样式文件

## 论文列表

已发表 10 篇期刊论文，涵盖以下研究方向：

- 劳动力市场与工资差异
- 人力资本与教育经济
- 政策效应评估
- 农村发展与扶贫
- 医疗卫生体制改革
- 数字金融与就业

## 部署方式

本项目部署在 GitHub Pages：

- 网址：https://sun0720.github.io/my-blog/

## 本地开发

如需在本地运行：

1. 克隆仓库：
```bash
git clone https://github.com/sun0720/my-blog.git
cd my-blog
```

2. 直接在浏览器中打开 `index.html` 即可预览

## 技术栈

- HTML5
- CSS3
- 纯静态网站，无需后端
